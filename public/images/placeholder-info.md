# NovaNews Image Placeholders

This directory contains placeholder images for the blog while the automation system generates real images.

## Required Images:

### Welcome/Featured
- welcome-novnews.jpg (1200x600) - Welcome post featured image
- featured-placeholder.jpg (1200x600) - Default featured image

### Category Placeholders
- ai-workplace.jpg (800x400) - AI/workplace themed
- quantum-chip.jpg (800x400) - Quantum computing
- ai-hacking.jpg (800x400) - Cybersecurity
- default-tech.jpg (800x400) - General tech

### Article Placeholders (rotating)
- placeholder-1.jpg through placeholder-4.jpg (600x300) 

## Image Generation
The automation system will generate relevant images for each post using Pollinations.ai.
Fallback to these placeholders if image generation fails.

## Style Guidelines
- Professional tech aesthetic
- Modern, clean design
- Dark/blue color scheme to match NovaNews branding
- No text overlays (handled by CSS)
- High quality, optimized for web
